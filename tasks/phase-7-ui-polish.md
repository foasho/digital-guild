# Phase 7: UIポリッシュ（美しさ向上）

**ステータス**: 未着手
**作成日**: 2026-01-01

## 概要
Phase 6完了後のUI確認で発見した、より美しくするための改善点をまとめる。
機能的な問題ではなく、視覚的な洗練度を高めることを目的とする。

---

## Worker UI改善

### Task 7.1: ギルド証カードのリデザイン

**現状の課題**:
- フラットすぎて存在感が薄い
- ランク表示が小さく目立たない

**改善案**:
- 背景にグラデーションまたは微細なパターンを追加
- ランクに応じたボーダーグロー効果を追加（Bronze/Silver/Gold/Platinum）
- アバター周りにリング状の装飾を追加
- JPYC残高の文字サイズを大きく、フォントウェイトを太く

**対象ファイル**:
- `frontend/components/worker/GuildCard.tsx`

---

### Task 7.2: ジョブカードのビジュアル強化

**現状の課題**:
- 「あなたにおすすめ」バッジのデザインが安っぽい
- カードの影が弱くフラット
- 報酬表示の「¥」記号が小さい

**改善案**:
- 「あなたにおすすめ」バッジをグラデーションリボンスタイルに変更
- カードにdrop-shadow-lgを適用し立体感を追加
- 報酬額のフォントをより大きく、太字に
- カード間のマージンを少し広げる（現在の詰まり感を解消）
- 画像のオーバーレイグラデーションを調整（下部をより暗く）

**対象ファイル**:
- `frontend/components/worker/JobCard.tsx`
- `frontend/app/worker/request-boards/page.tsx`

---

### Task 7.3: フッターナビゲーションの改善

**現状の課題**:
- アイコンが小さく視認性が低い
- 選択状態の視覚的フィードバックが弱い

**改善案**:
- アイコンサイズを24pxから28pxに拡大
- 選択中のタブに背景色（amber/gold系）を追加
- アイコン下のラベルのフォントサイズを少し大きく
- フッター全体の高さを少し増やす（py-2 → py-3）

**対象ファイル**:
- `frontend/components/layout/Footer.tsx`

---

### Task 7.4: マップページの修正

**現状の課題**:
- Leafletマップが表示されていない（背景画像のみ表示）
- 検索UIの視認性が低い

**改善案**:
- マップのz-index問題を調査・修正
- 検索ボタンにパルスアニメーションを追加
- マップピンのデザインをより魅力的に（影、ホバーエフェクト追加）

**対象ファイル**:
- `frontend/app/worker/request-map/page.tsx`
- `frontend/components/worker/MapView.tsx`

---

### Task 7.5: ジョブステータスバッジの視認性向上

**現状の課題**:
- 「着手中」バッジが背景画像と同化して見づらい
- バッジのコントラストが不足

**改善案**:
- バッジに白い縁取り（ring）を追加
- 背景色をより鮮明に（opacity-90以上）
- バッジサイズを少し大きく

**対象ファイル**:
- `frontend/app/worker/jobs/page.tsx`
- `frontend/components/worker/MyJobCard.tsx`

---

### Task 7.6: ウォレットカードの立体感追加

**現状の課題**:
- カードがフラットで質感が薄い
- 取引履歴の間隔が詰まりすぎ

**改善案**:
- ウォレットカードに微細なグラデーションと影を追加
- カード角にわずかな光沢効果
- 取引履歴の各項目間にdividerまたはマージン追加
- プラス金額を緑、マイナス金額を赤でより鮮明に

**対象ファイル**:
- `frontend/app/worker/wallet/page.tsx`

---

## Requester UI改善

### Task 7.7: 統計カードの情報整理

**現状の課題**:
- カードが小さく情報が詰まりすぎ
- 数字の視認性が低い

**改善案**:
- カードのパディングを増やす
- 数字のフォントサイズを大きく（text-2xl → text-3xl）
- アイコンに背景色（円形）を追加して目立たせる
- ラベルと数字の間隔を調整

**対象ファイル**:
- `frontend/components/requester/StatsSummary.tsx`

---

### Task 7.8: ダッシュボードジョブカードへの画像追加

**現状の課題**:
- ジョブカードに画像がなく、テキストのみで味気ない
- 他のページ（Worker側）との一貫性がない

**改善案**:
- RequesterJobCardにサムネイル画像を追加
- 左側にサムネイル、右側にテキスト情報のレイアウト
- 画像サイズは60x60〜80x80程度の正方形

**対象ファイル**:
- `frontend/components/requester/RequesterJobCard.tsx`

---

### Task 7.9: タブデザインの改善

**現状の課題**:
- タブがシンプルすぎて選択状態がわかりにくい
- 下線が細い

**改善案**:
- 選択中タブの背景色を追加（sky-100など）
- 下線を太く（2px → 3px）
- タブ間のマージンを調整
- ホバー時のフィードバックを強化

**対象ファイル**:
- `frontend/app/requester/dashboard/page.tsx`

---

### Task 7.10: ジョブ作成フォームの区切り明確化

**現状の課題**:
- フォームの各セクションがまとまりなく続いている
- タグのチェックボックスが小さい
- 送信ボタンが目立たない

**改善案**:
- フォームをセクションごとにグループ化（基本情報、詳細設定、チェックリスト）
- 各セクションにヘッダーと軽い区切り線を追加
- タグチェックボックスを大きく、押しやすく
- 送信ボタンをより目立つデザインに（大きく、グラデーション追加）

**対象ファイル**:
- `frontend/app/requester/jobs/new/page.tsx`

---

### Task 7.11: 応募者カードのリッチ化

**現状の課題**:
- 応募者情報がシンプルすぎる
- Trust Scoreの視覚的表現が弱い

**改善案**:
- Trust Scoreをプログレスバーまたはゲージで視覚化
- ランクバッジにグラデーション追加
- 承認/未承認状態のバッジをより目立たせる
- アバター周りに枠線追加

**対象ファイル**:
- `frontend/app/requester/jobs/[job_id]/page.tsx`

---

## 共通改善

### Task 7.12: アイコン画像の最適化

**現状の課題**:
- コンソールに`Image with src "/icons/*.png" has either width or height modified`警告が出ている

**改善案**:
- 全アイコン画像のwidth/heightを両方指定
- または`fill`プロパティを使用
- 画像サイズの一貫性を確保

**対象ファイル**:
- `frontend/components/layout/Footer.tsx`
- その他アイコンを使用しているコンポーネント

---

### Task 7.13: ローディング状態のアニメーション追加

**現状の課題**:
- データロード中の表示が味気ない

**改善案**:
- スケルトンローディングの追加
- スピナーにブランドカラーを適用
- フェードインアニメーションの追加

**対象ファイル**:
- 各ページコンポーネント
